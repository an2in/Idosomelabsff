# CASE-002-M57Jean — Data Exfiltration Incident

## Portfolio summary

This investigation examines a data exfiltration incident involving the user Jean at M57. By analyzing the `outlook.pst` extracted from Jean's disk images using FTK Imager and viewing it with GoldFynch, the investigation identified a successful social engineering attack. An external party (`tuckgorge@gmail.com`) impersonated an executive (Alison) and tricked Jean into sending sensitive company information. The main uncertainty remaining is how the external attacker obtained prior knowledge of the internal discussion.
