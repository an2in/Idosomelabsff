# 000 — The Suspicious Misplaced Notes

## Portfolio summary

This investigation examined three supplied plain text files exported from a training workstation to identify restricted project information and analyze potential duplicates. The evidence set consisted of a meeting notes file (E-001) and two review notes (E-002 and E-003). The analysis confirmed that E-002 and E-003 contain Project Nightjar information explicitly marked for restricted distribution, and are exact byte-for-byte duplicates. However, because of the complete absence of OS logs, registry records, or original filesystem metadata, it is impossible to determine who created/copied the files or which duplicate existed first.
